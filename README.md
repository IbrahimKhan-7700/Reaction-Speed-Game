# Reaction-Speed-Game 🎯
**Description:** A reaction speed game built using an Arduino Uno R3. It uses buttons, a active buzzer, LEDs to test a player reaction time against another player.

## Things Used
| Component | Quantity | Note |
| --------- | -------- | ---- |
| Active Buzzer| 1 |
| Green LED | 1 |
| Yellow LED | 1 |
| Red LED | 1 | |
| White LED | 2 |
| Buttons | 2 |
| Resistor | 5 | 220 Ohms |
| Breadboard | 1 |
| Jumper Wires | 14 |

2. Wire the components
